# Debiteuren Monitoring Applicatie

Een Flask-gebaseerde webapplicatie voor het monitoren van debiteuren en bedrijfsgegevens.

## Features

- Bedrijfsgegevens opzoeken via BTW-nummer of bedrijfsnaam
- Batch management voor debiteuren
- PDF export van batches
- CSV upload functionaliteit
- Dashboard met zoekfunctionaliteit

## Setup

1. Clone deze repository
2. Installeer dependencies: `pip install -r requirements.txt`
3. Run database migrations: `flask db upgrade`
4. Start de applicatie: `python run.py`
